# Project 02 — Frontend (Starter)

React + Vite + TypeScript + Tailwind + React Router + TanStack Query + Zustand.

## Setup

```bash
npm install
cp .env.example .env   # Set VITE_API_URL to your API (e.g. http://localhost:3000)
npm run dev
```

Run the Project 02 API starter in another terminal so the frontend can fetch data.
